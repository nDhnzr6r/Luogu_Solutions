# P1075 [NOIP 2012 普及组] 质因数分解

## 题目描述

已知正整数 $n$ 是两个不同的质数的乘积，试求出两者中较大的那个质数。

## 输入格式

输入一个正整数 $n$。

## 输出格式

输出一个正整数 $p$，即较大的那个质数。

## 输入输出样例 #1

### 输入 #1

```
21
```

### 输出 #1

```
7
```

## 说明/提示

$1 \le n\le 2\times 10^9$。

NOIP 2012 普及组 第一题

---

# 个人做题思路

## 核心思路

已知 n 是两个不同质数的乘积，枚举较小的质因数 i，判断 i 和 n/i 是否均为质数，输出较大者。

## 具体步骤

1. 读入 n
2. 枚举 i 从 2 到 sqrt(n)，用 6k±1 法则快速筛掉合数
3. 当 i 为质数且 n % i == 0 时，计算 b = n / i
4. 同样用 6k±1 法则验证 b 是否为质数
5. 若两者均为质数，输出 max(i, b) 即较大者

## 踩坑点

- 同时验证 i 和 n/i 两个因子均为质数，因为 n 可能被某个合数整除（但本题中 n 只有两个质因子，理论上第一个整除 n 的数就是那个较小的质数，无需验证 i 的质数性）
- 1LL * i * i <= n 防止 int 溢出
- 6k±1 法则是质数判定的经典优化，但本题数据范围下暴力 sqrt 枚举也足够通过

## 更简洁的解法

由于 n 只有两个质因子，只需从小到大枚举找到第一个整除 n 的数即为较小的质数 p，答案 = n / p。无需同时验证两个因子的质数性。
