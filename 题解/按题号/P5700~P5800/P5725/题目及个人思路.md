# P5725 【深基4.习8】求三角形

## 题目描述

模仿例题，打印出不同方向的正方形，然后打印三角形矩阵。中间有个空行。

## 输入格式

输入矩阵的规模，不超过 $9$。

## 输出格式

输出矩形和三角形。

## 输入输出样例 #1

### 输入 #1

```
4
```

### 输出 #1

```
01020304
05060708
09101112
13141516

      01
    0203
  040506
07080910
```

---

# 个人做题思路

## 核心思路

用全局自增变量 a 控制数字编号，分两部分打印：正方形矩阵和右对齐三角形，中间空行隔开。

## 具体步骤

1. 读入规模 x，a 从 1 开始
2. 第一部分：双层循环打印 x×x 正方形，每位占两字符（不足两位补 0），a 自增
3. 输出空行分隔
4. 重置 a = 1
5. 第二部分：双层循环打印三角形
   - 外层 i 控制行数（0 到 x-1）
   - 内层先打印 `x - i - 1` 组空格（每组两个空格，实现右对齐）
   - 再打印 `i + 1` 个两位数字
6. 同样不足两位补零

## 踩坑点

- 每行数字用两位输出，第二位用 a 自增，注意区分补零和不补零的情况
- 三角形部分每行递增一个数字，对齐空格数为 `(x - i - 1) × 2`
- a 在第二部分开始前要重置为 1
