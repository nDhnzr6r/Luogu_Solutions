# P5726 【深基4.习9】打分

## 题目描述

现在有 $n(n \le 1000)$ 位评委给选手打分，分值从 $0$ 到 $10$。需要去掉一个最高分，去掉一个最低分（如果有多个最高或者最低分，也只需要去掉一个），剩下的评分的平均数就是这位选手的得分。现在输入评委人数和他们的打分，请输出选手的最后得分，精确到 $2$ 位小数。

## 输入格式

第一行输入一个正整数 $n$，表示有 $n$ 个评委。

第二行输入 $n$ 个整数，第 $i$ 个整数表示第 $i$ 个评委打出的分值。

## 输出格式

输出一行一个两位小数，表示选手的最后得分。

## 输入输出样例 #1

### 输入 #1

```
5
9 5 6 8 9
```

### 输出 #1

```
7.67
```

## 说明/提示

数据保证，$3 \leq n \leq 1000$，每个评委打出的分值为 $0$ 到 $10$（含 $0$ 与 $10$）之间的整数。

---

# 个人做题思路

## 核心思路

遍历一次数组找出最高分和最低分的位置，求和时跳过这两个位置，最后算平均值。

## 具体步骤

1. 读入 n 和 n 个分数，同时记录最小值 mi 及其下标 mix、最大值 mx 及其下标 mxx
2. 遍历数组求和，遇到 mix 或 mxx 时跳过
3. 输出 `sum / (n - 2)`，保留两位小数

## 踩坑点

- 只去掉一个最高分和一个最低分（而不是去掉所有同分的），所以记录下标而非值来判断是否跳过
- 初始值：mi 设 100（最大可能分值），mx 设 -1
- 输出保留两位小数用 `fixed << setprecision(2)`
