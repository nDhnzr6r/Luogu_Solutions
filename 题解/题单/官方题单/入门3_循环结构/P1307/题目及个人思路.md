# P1307 [NOIP 2011 普及组] 数字反转

## 题目描述

给定一个整数 $N$，请将该数各个位上数字反转得到一个新数。新数也应满足整数的常见形式，即除非给定的原数为零，否则反转后得到的新数的最高位数字不应为零（参见样例 2）。

## 输入格式

一个整数 $N$。

## 输出格式

一个整数，表示反转后的新数。

## 输入输出样例 #1

### 输入 #1

```
123
```

### 输出 #1

```
321
```

## 输入输出样例 #2

### 输入 #2

```
-380
```

### 输出 #2

```
-83
```

## 说明/提示

**【数据范围】**

$-1,000,000,000\leq N\leq 1,000,000,000 $。

noip2011 普及组第一题

---

# 个人做题思路

## 核心思路

转字符串反转，分正数、负数、零三种情况处理。

## 具体步骤

1. 读入 n
2. 正数：去掉末尾 0 后，`to_string` 转字符串再 `reverse` 输出
3. 负数：同样去掉末尾 0，反转后用 `erase` 删掉末尾的负号，先输出 `"-"` 再输出反转结果
4. n=0：直接输出 0

## 踩坑点

- 负数转字符串后反转，负号会跑到字符串末尾，需要 `erase` 去掉再手动加回开头
- 末尾有 0 的情况（如 -380）需先用 while 循环去零再反转
- N 范围到 $10^9$，int 足够
