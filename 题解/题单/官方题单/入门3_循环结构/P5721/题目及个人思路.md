# P5721 【深基4.例6】数字直角三角形

## 题目描述

给出 $n$，请输出一个直角边长度是 $n$ 的数字直角三角形。所有数字都是 $2$ 位组成的，如果没有 $2$ 位则加上前导 $0$。

## 输入格式

输入一个正整数 $n$。

## 输出格式

输出如题目要求的数字直角三角形。

## 输入输出样例 #1

### 输入 #1

```
5
```

### 输出 #1

```
0102030405
06070809
101112
1314
15
```

## 说明/提示

数据保证，$1\le n\le13$。

---

# 个人做题思路

## 核心思路

双层循环控制行列，外层行数递减，内层输出数字并补前导零。

## 具体步骤

1. 读入 n，初始化计数器 `x = 1`
2. 外层循环 i 从 0 到 n-1，控制行数
3. 内层循环 j 从 i 到 n-1，每行输出 n-i 个数
4. 输出时判断 `x / 10 == 0`，小于 10 则补 `0`
5. x 自增，每行结束后换行

## 踩坑点

- 两位数判断条件是 `x / 10 == 0`，不是 `x < 10`（效果相同但代码写法不同）
- 内层循环起点是 `j = i`，实现每行少输出一个数
- 前导零直接在前面输出 `"0"` 即可
