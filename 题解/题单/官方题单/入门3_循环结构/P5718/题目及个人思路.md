# P5718 【深基4.例2】找最小值

## 题目描述

给出 $n$ 和 $n$ 个整数 $a_i$，求这 $n$ 个整数中最小值是什么。

## 输入格式

第一行输入一个正整数 $n$，表示数字个数。

第二行输入 $n$ 个非负整数，表示 $a_1,a_2 \dots a_n$，以空格隔开。

## 输出格式

输出一个非负整数，表示这 $n$ 个非负整数中的最小值。

## 输入输出样例 #1

### 输入 #1

```
5
5 7 4 2 6
```

### 输出 #1

```
2
```

## 说明/提示

数据保证，$n\le100$ 且 $0\le a_i \le 1000$。

---

# 个人做题思路

## 核心思路

遍历数组找最小值，属于循环结构的基础应用。

## 具体步骤

1. 读入 n 和 n 个整数到数组 `a[105]`
2. 设初始最小值 `t = 999999`，作为哨兵值
3. 遍历数组，逐个比较更新最小值
4. 输出 `t`

## 踩坑点

- 初始最小值要设得足够大（$a_i \le 1000$，设 999999 够用）
- 数组大小 >= 100（$n \le 100$）
