# P5719 【深基4.例3】分类平均

## 题目描述

给定 $n$ 和 $k$，将从 1 到 $n$ 之间的所有正整数可以分为两类：A 类数可以被 $k$ 整除（也就是说是 $k$ 的倍数），而 B 类数不能。请输出这两类数的平均数，精确到小数点后 $1$ 位，用空格隔开。

数据保证两类数的个数都不会是 $0$。

## 输入格式

输入两个正整数 $n$ 与 $k$。

## 输出格式

输出一行，两个实数，分别表示 A 类数与 B 类数的平均数。精确到小数点后一位。

## 输入输出样例 #1

### 输入 #1

```
100 16
```

### 输出 #1

```
56.0 50.1
```

## 说明/提示

数据保证，$1 \leq n\leq 10000$，$1 \leq k \leq 100$。

---

# 个人做题思路

## 核心思路

遍历 1~n，按能否被 k 整除分成两类，分别统计个数与总和，最后求平均值。

## 具体步骤

1. 读入 n、k
2. 遍历 i 从 1 到 n，判断 `i % k`
   - 能整除：计入 A 类（计数+1，总和累加）
   - 不能整除：计入 B 类
3. 用 `(double)a / ai` 计算 A 类平均值，B 类同理
4. `fixed << setprecision(1)` 保留一位小数输出

## 踩坑点

- 平均值计算前需将分子或分母转为 double，否则整数除法会丢失小数
- 输出保留一位小数用 `fixed << setprecision(1)`
