# P5727 【深基5.例3】冰雹猜想

## 题目描述

给出一个正整数 $n$，然后对这个数字一直进行下面的操作：如果这个数字是奇数，那么将其乘 $3$ 再加 $1$，否则除以 $2$。经过若干次循环后，最终都会回到 $1$。经过验证很大的数字（$7\times10^{11}$）都可以按照这样的方式比变成 $1$，所以被称为"冰雹猜想"。例如当 $n$ 是 $20$，变化的过程是 $20\to 10\to 5\to 16\to 8\to 4\to 2\to 1$。

根据给定的数字，验证这个猜想，并从最后的 $1$ 开始，倒序输出整个变化序列。

## 输入格式

输入一个正整数 $n$。

## 输出格式

输出若干个由空格隔开的正整数，表示从最后的 $1$ 开始倒序的变化数列。

## 输入输出样例 #1

### 输入 #1

```
20
```

### 输出 #1

```
1 2 4 8 16 5 10 20
```

## 说明/提示

数据保证，$1 \le n\le 100$。

---

# 个人做题思路

## 核心思路

模拟冰雹猜想过程，用数组按顺序存储每一步的结果，最后倒序输出。

## 具体步骤

1. 特判 n=1，直接输出 1
2. 用数组 a 存储变化序列，a[0] = n
3. 循环：根据 a[i-1] 的奇偶性计算 a[i]，直到 a[i] == 1
4. 从 i 到 0 倒序输出

## 踩坑点

- n=1 时没有变化过程，需要特判
- 数组大小要足够：n ≤ 100 时，变化次数最多约 100 次左右，开 10000 很安全
- 倒序从 i 开始（包含最后的 1），到 0 结束
