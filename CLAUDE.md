# CLAUDE.md — Luogu_Solutions

## 项目概述
个人洛谷刷题记录与题解整理仓库，每道题同时按题单和题号范围双索引存放。

## 目录规范
```
题解/
├── 题单/{题单来源}/{题单名}/{题号}/
│   ├── {题号}.cpp          # AC 代码（带简洁注释）
│   └── 题目及个人思路.md    # 题目描述 + 个人做题思路
└── 按题号/P{起点}~P{终点}/{题号}/
    └── （同上，按题号范围索引）
```

## 新增题解时

1. 在 `题解/题单/{来源}/{题单名}/{题号}/` 下创建文件夹
2. 创建 `{题号}.cpp`，将我写好的代码拷贝进去，一般在"C:\Users\nDhnzr6r\Desktop\做题区.cpp"中
3. 创建 `题目及个人思路.md` ，将题目从`题目.md`复制到 `题目及个人思路.md` 中
4. 打上分割线，添加 `# 个人做题思路`，根据我的解法补全个人做题思路部分
5. 在代码中增加简单注释
6. 根据题号范围，复制到 `题解/按题号/P{起点}~P{终点}/{题号}/` 下
7. 更新 README.md 的进度表，目录结构中每个文件夹只写一道题作代表，其余用 `...` 省略

## 文档风格

- 所有 MD 文件不要使用加粗样式（`**...**`），保持简洁纯文本

## 推送

- 新增题解后不要自动推送，等待我审核完毕说"推送"后再推
- commit message 格式：`P{题号}: {简述}`
- push 前确保通过代理访问 GitHub
